# Data Analysis of Car Crashes in California
Various plots of car crashes in California to practice using Tableau. Data was cleaned using Python, then sent to Tableau for plotting.

[Data](https://www.kaggle.com/sobhanmoosavi/us-accidents)  

[Python/Notebook Version](https://github.com/carrnick/Car-Accident-Data-Analysis/blob/main/California.ipynb)
